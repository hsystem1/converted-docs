**Como acessar o HPRICE**

**Processo:**

Entrar no site -> Login -> Código (se tiver 2FA) -> Selecionar hotel

**Entrar no HPRICE**

* Acesse a URL do HPRICE: Digite no seu navegador (Google, Chrome, Firefox, etc.) o endereço de login da plataforma HPRICE.
* Link: [HPrice](https://hprice.hsystem.com.br/login)
* Tela Inicial: A tela inicial do HPRICE será exibida.
* Clique em "Login Único" botão laranja para prosseguir.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_102523_fe79/images/Como_acessar_o__HPRI_image2_f4a904e7.png)

### **2.Fazer Login**

Você será redirecionado para a tela de login do **HSystem**, onde usará suas credenciais:

* Veja o campo Endereço de E-mail
* Digite seu e-mail
* Veja o campo SENHA (Logo abaixo)
* Digite sua senha
* Clique no botão azul **Entrar**
* ***Caso tenha esquecido sua senha, use a opção "Esqueceu sua senha?".***

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_102523_fe79/images/Como_acessar_o__HPRI_image3_84092512.png)

### **3. Código de Autenticação (Se você tiver 2FA)**

Importante! Este passo só aparece se você tem autenticação de dois fatores ativada

* A tela vai pedir ‘Código do Autenticador’
* Abra o aplicativo autenticador (Google, Authenticator, Microsoft Authenticator, etc.)
* Procure o código de 6 dígitos e digite na tela
* Clique no botão azul Entrar

**Observação:** Se você já esta logado em outro sistema Hsystem (Hbook ou Hunit , pode ser que o código não seja pedido. Neste caso, você vai direto para a próxima tela.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_102523_fe79/images/Como_acessar_o__HPRI_image5_9815b3f8.png)

### **4. Selecionar o Hotel/Propriedade**

* Ao acessar o **HPRICE**, no canto direito clique e **"Selecione seu hotel"**.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_102523_fe79/images/Como_acessar_o__HPRI_image_268c4b52.png)

* Veja a barra de pesquisa **(Escrito Pesquisar Hotel ou ID)**
* Digite o nome do seu hotel **OU o ID de integração (número)**
* Seu hotel vai aparecer a lista abaixo
* Clique no círculo ao lado do nome do hotel (para selecionar)
* Clique no botão laranja **(Selecionar Hotel)**
* Não precisa fazer login novamente

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_102523_fe79/images/Como_acessar_o__HPRI_image4_0dc32e63.png)

**5. Como trocar de hotel**

* Se você precisar mudar de hotel ou propriedade dentro do HPRICE, não é necessário fazer um novo login.
* **Localize o Seletor:** No canto superior da tela, procure o nome da propriedade que está ativa no momento
* **Troque de Hotel:** Clique no nome da propriedade ou na seta ao lado. Isso deve reabrir a tela de **"Selecione seu hotel"**, permitindo que você escolha outra unidade caso possua.

**Resumo**

**Não lembro meu e-mail**

* Entre em contato com o suporte@hsystem.com.br

**Não lembro minha senha**

* Clique em ‘Esqueceu sua senha? Na tela de login

**Código do autenticador não funciona**

* Confira se o relógio do celular está certo – Aguarde novo código aparecer – Use o código novo

Não acho meu hotel na lista

* Tente digitar só parte do nome – Ou use o ID de integração (número) - Se não encontrar, contate o suporte.

**Glossário**

|  |  |
| --- | --- |
| **Termo** | **Significa** |
| Login Único | Botão para entrar usando mesma conta da Hsystem |
| 2 FA | Autenticação no celular que gera códigos de segurança |
| Autenticador | Aplicativo no celular que gera códigos de segurança |
| ID de integração | Número do hotel no sistema |
| Navegador | Programa para acessar internet (Chrome, Firefox, Edge) |
| Sessão | Tempo que voce fica logado no sistema |
