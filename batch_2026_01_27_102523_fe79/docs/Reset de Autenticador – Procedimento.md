**Reset de Autenticador**

**1. Sobre o Procedimento de reset de autenticador**

O reset de autenticador é necessário quando você não consegue mais acessar o sistema usando o código de autenticação em dois fatores.

Antes de solicitar o reset, é importante identificar o motivo do problema, pois o processo pode variar dependendo da situação.

**2. Identificar o Motivo da Solicitação**

Primeiro, identifique qual é a sua situação:

**Situação 1: Você Trocou de Celular ou Aparelho**

Se você trocou de celular ou perdeu o aparelho onde estava instalado o autenticador, podemos seguir diretamente com o processo de reset.

Neste caso, não é necessário fazer outras verificações. Vá direto para a seção 4 deste documento.

**Situação 2: Você Está Com Problema ao Validar o Código**

Se você ainda tem o aparelho com o autenticador, mas o código não está funcionando, **antes de solicitar o reset**, verifique os itens abaixo:

**3. Verificações Antes de Solicitar o Reset**

Se você está tendo problemas para validar o código do autenticador, faça estas verificações:

**Verificação 1: Data e Hora dos Aparelhos**

Seu celular e computador estão com a **mesma data e hora**, inclusive nos minutos?

* Verifique se ambos os aparelhos estão sincronizados
* Verifique se o ajuste de data/hora automático está ativado em ambos
* Se não estiverem sincronizados, ajuste e tente novamente

**Por que isso é importante:** O código do autenticador é baseado no horário. Se os aparelhos estiverem com horários diferentes, o código não vai funcionar.

**Verificação 2: Tempo de Validade do Código**

O código está sendo digitado **sem espaços** e **dentro dos 30 segundos** de validade?

* Não inclua espaços ao digitar o código
* Digite o código rapidamente, antes que ele mude
* Aguarde um novo código se o tempo estiver acabando

**Por que isso é importante:** Cada código tem validade de apenas 30 segundos. Depois desse tempo, um novo código é gerado automaticamente.

**Verificação 3: Autenticador Correto**

Se você tem **mais de um autenticador instalado** no seu celular, está usando o autenticador correspondente ao **seu login correto**?

* Verifique se você está usando o código da conta certa
* Se tem várias contas, confirme qual é a que corresponde ao login que está usando

**Por que isso é importante:** Muitas pessoas têm múltiplas contas no autenticador. É preciso usar o código da conta correta.

**4. Como Solicitar o Reset (Passo a Passo)**

Se você verificou todos os itens acima e mesmo assim não consegue autenticar, ou se você trocou de celular, siga o processo abaixo:

**Passo 1: Preparar as Informações**

Antes de enviar o e-mail, tenha em mãos:

* Seu nome completo
* Seu e-mail de login no sistema
* O motivo da solicitação (troca de celular ou problema de validação)

**Passo 2: Enviar E-mail com a Solicitação**

Envie um e-mail para: **suporte@hsystem.com.br**

**Importante:** O e-mail deve ser enviado **pelo endereço do contratante**, ou seja, aquele que está cadastrado em nosso sistema.

**Configure o e-mail assim:**

* **Destinatário:** suporte@hsystem.com.br
* **Assunto:** Solicitação de Reset de Autenticador | Nome do Hotel
* **Corpo do e-mail:**
  + Nome completo
  + E-mail de login
  + Motivo da solicitação

**Exemplo de e-mail:**

Assunto: Solicitação de Reset de Autenticador | Nome do Hotel

Prezada equipe,

Solicito o reset do meu autenticador.

Nome completo: João da Silva
E-mail de login: joao.silva@hotel.com.brMotivo: Troquei de celular e não consigo mais acessar o autenticador anterior

Aguardo retorno.

**Passo 3: Aguardar Confirmação por Telefone**

Após recebermos o e-mail, nossa equipe fará uma **validação de segurança por telefone** com o contratante para confirmar os dados.

**Como funciona:**

* Nossa equipe entrará em contato no telefone registrado
* O contratante precisará confirmar a solicitação
* Faremos perguntas de segurança para validar a identidade
* Somente após essa validação faremos o reset do autenticador

**Importante:** Mantenha-se disponível para atender a ligação de segurança.

**Passo 4: Configurar Novo Autenticador**

Após a validação e o reset, você receberá instruções por e-mail sobre como configurar o autenticador novamente no seu aparelho.

**5. Dicas Importantes**

* **Sempre use o e-mail do contratante** para fazer a solicitação
* **Verifique data e hora** antes de solicitar reset por problema de código
* **Mantenha-se disponível** para atender a ligação de validação
* **Anote seus códigos de backup** quando configurar o novo autenticador
* **Não compartilhe** os códigos do autenticador com outras pessoas

**6. Por Que o Reset Precisa de Validação**

A validação por telefone é uma **medida de segurança** essencial para:

* Confirmar que a solicitação é legítima
* Proteger sua conta contra acessos não autorizados
* Garantir que apenas o responsável legal possa fazer alterações
* Prevenir fraudes e tentativas de invasão

Este processo pode parecer trabalhoso, mas é fundamental para manter seus dados seguros.

## **Glossário de Termos**

|  |  |
| --- | --- |
| **Termo** | **Definição** |
| **Autenticador** | Aplicativo instalado no celular que gera códigos temporários de 6 dígitos para validação de login. |
| **Reset de Autenticador** | Processo de desvinculação e reconfiguração do autenticador quando o usuário perde acesso aos códigos. |
| **Autenticação em Dois Fatores** | Método de segurança que exige senha + código temporário do autenticador para fazer login. |
| **Código de Autenticação** | Sequência de 6 dígitos gerada pelo autenticador, válida por 30 segundos. |
| **Contratante** | Pessoa ou empresa que assinou o contrato de serviços e é o responsável legal pela conta. |
| **E-mail de Login** | Endereço de e-mail usado para entrar no sistema. |
| **Validação de Segurança** | Processo de confirmação da identidade do solicitante através de ligação telefônica. |
| **Data e Hora Sincronizadas** | Ajuste automático de data e hora para garantir que celular e computador estejam no mesmo horário. |
| **Tempo de Validade** | Período de 30 segundos em que cada código do autenticador é válido. |
| **Códigos de Backup** | Códigos alternativos fornecidos durante a configuração inicial do autenticador para uso emergencial. |
| **Troca de Celular** | Situação em que o usuário adquire um novo aparelho e perde acesso ao autenticador do aparelho anterior. |
| **Login** | Processo de entrar no sistema usando e-mail e senha. |
| **Validação por Telefone** | Ligação realizada pela equipe de suporte para confirmar a identidade do solicitante. |
| **Ajuste Automático de Hora** | Configuração do celular/computador que sincroniza automaticamente data e hora com servidores de internet. |
| **Aplicativo Autenticador** | Programa instalado no celular, como Google Authenticator, Microsoft Authenticator, Authy, etc. |
