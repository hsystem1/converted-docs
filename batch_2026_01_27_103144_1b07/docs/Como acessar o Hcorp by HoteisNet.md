**Como acessar o Hcorp/Hoteisnet**

### **1.Passo a Passo para Acesso ao Hunit (Gestor de Canais)**

Hcorp não tem sistema próprio. Você entra pelo HUNIT:

* **Acesse a URL do Hunit:** Digite no seu navegador o endereço de login do **Hunit.**
* **Link:** [Login - HUNIT Channel Manager](https://extranet.hunit.com.br/HUNIT/Account/Login)
* **Tela Inicial:** Na tela inicial do **Hunit,** clique em **"LOGIN ÚNICO"**.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_103144_1b07/images/Como_acessar_o_Hcorp_image_cec96294.png)

**2. Fazer Login**

* **Preencha seus Dados:** Você será redirecionado para a tela de login do H System. Digite seu **Endereço de e-mail** e sua **Senha**.
* **Clique em "Entrar"**

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_103144_1b07/images/Como_acessar_o_Hcorp_image6_3872a59c.png)

#### **3.Segurança e Autenticação**

* **Código do Autenticador:** Se a autenticação de dois fatores estiver ativada, insira o código gerado pelo seu aplicativo autenticador.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_103144_1b07/images/Como_acessar_o_Hcorp_image3_7c5e073c.png)

**Observação:** Se você já estiver logado em outros produtos da **HSystem**, o código do Autenticador **não será solicitado**, permitindo o acesso direto.

**4.Selecione a Propriedade**

* **Pesquise e Selecione:** Use a barra de **"Pesquisar"** para encontrar o hotel que deseja administrar e clique no cartão da propriedade para acessá-la.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_103144_1b07/images/Como_acessar_o_Hcorp_image2_eb62e49f.png)

**5.Acessar o HCORP**

Após fazer o login e selecionar a propriedade no Hunit, o HCORP/HOTEISNET é acessado diretamente no menu superior:

* **Localize o Menu:** No menu superior do painel de gerenciamento, procure o item **"HCORP BY HOTÉISNET" e clique nele**.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_103144_1b07/images/Como_acessar_o_Hcorp_image4_d8c055e7.png)

* **Navegue nas Opções:** Você será direcionado para a área do Hcorp by Hoteisnet, onde poderá navegar entre as subseções, como **CONFIGURAÇÃO** e **CANAIS DE VENDA**.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_103144_1b07/images/Como_acessar_o_Hcorp_image5_cdfff720.png)

**Glossário**

|  |  |
| --- | --- |
| **Termo** | **Significado** |
| Hcorp | Sistema de gestão corporativa by hoteisnet |
| Hunit | Sistema gestor de canais |
| Login Único | Botão para entrar usando mesma conta da Hsystem |
| Menu superior | Barra de opções no topo da tela |
| Cartão | Retângulo clicável com o nome do hotel |
| Configuração | Seção do Hcorp para ajustes do sistema |
| Canais de venda | Seção do Hcorp para gerenciar onde você vende |
