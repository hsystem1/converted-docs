# **Aplicativo de Autenticação**

1. **O que é autenticação 2FA**

* 2FA Significa ‘Autenticação em Duas Etapas’: É um sistema de segurança que protege suas informações e o acesso aos sistemas da Hsystem.

* **Primeira etapa:** Você digita seu e-mail e senha (como sempre fez)
* **Segunda etapa:** Você precisa digitar um código que aparece no seu celular

***Por que duas etapas?***

*Mesmo que alguém descubra sua senha, essa pessoa* ***NÃO*** *conseguirá entrar sem o código do seu celular. Isso protege muito mais suas informações.*

## **3.Como funciona o aplicativo?**

O que o aplicativo faz:

* Gera um código de 6 números
* Esse código muda automaticamente em segundos
* O código é exclusivo para você e só funciona no seu celular

**Como você usa:**

* Abre o sistema no computador
* Digita seu e-mail e senha
* O sistema pede o código do autenticador
* Você pega seu celular
* Abre o aplicativo autenticador
* Vê o código de 6 números
* Digita esse código no computador
* Pronto! Você está dentro do sistema

**4. Regras de uso**

**Leia com atenção! Seguir essas regras evita problemas e bloqueios.**

## **O Autenticador É Individual**

O que significa:

• Cada pessoa deve ter SEU PRÓPRIO aplicativo configurado

• O código do João NÃO serve para a Maria

• Você não pode compartilhar seu código com outra pessoa

***Por que é assim?***

*É por segurança. Se várias pessoas usarem o mesmo código, fica impossível saber quem entrou no sistema.*

## **O Autenticador Fica Vinculado ao Celular**

O que significa:

• O código só funciona no celular onde você configurou

• Se você trocar de celular, o código antigo para de funcionar

• Você não consegue usar o mesmo código em dois celulares diferentes

***Exemplo:***

*João configurou no celular dele. Se João emprestar o celular para Maria, ela verá o código de João (mas isso não é recomendado!). Se João comprar um celular novo, o código do celular velho não funciona mais.*

## **Troca de Celular Precisa de Reset**

O que fazer se você vai trocar de celular:

**ANTES de trocar:**

* Entre em contato com o suporte da HSystem
* Informe que você vai trocar de celular
* Peça o reset do autenticador
* Aguarde o suporte fazer o reset
* Configure no celular novo

**O que fazer se você JÁ trocou sem avisar:**

* Entre em contato com o suporte
* Explique que trocou de celular
* Siga as orientações de segurança do suporte
* O suporte vai fazer o reset
* Configure no celular novo

**ATENÇÃO:**

Se você não pedir reset, você não consegue entrar no sistema! Por isso, SEMPRE avise o suporte antes de trocar de celular.

## **5.Horário do Celular e Computador Devem Estar Iguais**

**Por que o horário é importante:**

O código do aplicativo é gerado baseado na hora exata. Se o horário do seu celular estiver diferente do horário do computador, o código não vai funcionar.

Como garantir que está correto:

**No celular:**

* Vá em Configurações (ou Ajustes)
* Procure por 'Data e Hora' ou 'Data/Hora'
* Ative a opção 'Definir automaticamente' ou 'Automático'
* Deixe marcado para ajustar pela rede

**No computador:**

* Verifique o relógio no canto da tela
* Compare com o horário do celular
* Se estiver diferente, ajuste nas configurações do Windows ou Mac

**Atenção:**

Diferença de até 1 minuto já pode fazer o código não funcionar!

## **6. Use o Código Corretamente**

Como digitar o código:

**CERTO:** 123456 (sem espaços)

**ERRADO:** 123 456 (com espaço)

Outras dicas importantes:

• O código tem validade de 30 segundos

• Se você demorar muito, vai aparecer um código novo

• Nesse caso, use o código novo que está aparecendo

• Se você tiver várias contas no aplicativo, confira se está vendo o código da conta certa

## **7. Um Autenticador Por Usuário**

**O que NÃO fazer:**

* NÃO configure vários usuários no mesmo celular
* NÃO use o celular da empresa para todos os funcionários
* NÃO deixe vários códigos diferentes no mesmo aplicativo

**Por que não pode:**

• Fica confuso saber qual código usar

• Aumenta o risco de usar o código errado

• Pode causar bloqueio de acesso

**8. O ideal:**

* Cada pessoa tem seu celular -> Seu aplicativo -> Seu código

## **9. Quando pedir reset do autenticador**

Reset é quando o suporte 'zera' sua configuração para você poder configurar tudo de novo.

Você PRECISA pedir reset nestes casos:

**1. Trocou de celular**

* Comprou celular novo ou ganhou outro? Peça reset!

**2. Perdeu o celular**

* Celular roubado, perdido ou quebrado? Peça reset urgente!

**3. Reinstalou o aplicativo**

* Desinstalou e instalou de novo? Peça reset!

**4. Formatou o celular**

* Voltou o celular para configurações de fábrica? Peça reset!

**5. Código não funciona**

* Horário está certo, código sem espaço, mas ainda não entra? Peça reset!

Como pedir reset:

* Entre em contato com o suporte da HSystem
* Informe seu nome completo e e-mail cadastrado
* Explique o motivo do reset (trocou celular, perdeu, etc.)
* O suporte vai fazer verificações de segurança
* Responda as perguntas de segurança
* Aguarde o suporte fazer o reset
* Configure o autenticador de novo no seu celular

**6. Por que essas regras existem**

As regras não existem para dificultar sua vida. Elas existem para PROTEGER você e o hotel.

Proteção contra:

* **Roubo de informações -** Pessoas mal-intencionadas não conseguem entrar no sistema
* **Acesso indevido -** Só você consegue entrar com sua conta
* **Fraudes -** Evita que alguém faça algo errado usando seu nome
* **Vazamento de dados -** Informações dos hóspedes ficam protegidas

***Lembre-se:***

*Quanto mais seguro o sistema, mais protegidos estão seus dados, os dados dos hóspedes e as informações financeiras do hotel.*

**Sempre que possível, peça para criar usuários individuais. Evita problemas, é mais seguro e mais prático no dia a dia!**

# **7. Como resolver problemas comuns**

**Problema 1: Código não funciona**

**O que fazer:**

* Verifique o horário do celular (deve estar em automático)
* Verifique se digitou sem espaços
* Espere aparecer um código novo e tente de novo
* Se ainda não funcionar, peça reset ao suporte

**Problema 2: Perdi meu celular**

**O que fazer:**

* Entre em contato com o suporte IMEDIATAMENTE
* Informe que perdeu o celular
* Peça o reset do autenticador
* Configure em um celular novo

**Problema 3: Aparece 'código inválido'**

**Causas possíveis:**

* Horário do celular diferente do computador
* Você digitou com espaço
* O código já expirou (passou dos 30 segundos)
* Você está vendo o código de outra conta no aplicativo

**Problema 4: Vários códigos no aplicativo**

**O que fazer:**

* Identifique qual é o código do sistema HSystem
* Geralmente aparece o nome 'HSystem' ou seu e-mail
* Use APENAS esse código
* Ignore os outros códigos (são de outros sistemas)

**Resumo Final:**

Cada pessoa = 1 celular = 1 aplicativo = 1 código

Vai trocar de celular = avise o suporte ANTES

Horário automático SEMPRE ligado

Código sem espaço + usar em 30 segundos

Problema? Contate o suporte imediatamente
