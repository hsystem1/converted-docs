**Como criar uma tarifa**

Orientações para mapeamento

Para que possamos configurar corretamente, preciso que primeiro você realize algumas etapas nos canais de venda.

**1. O que é uma tarifa?**

Tarifa é o preço que você cobra pela hospedagem. Você pode ter tarifas diferentes para situações diferentes, como: tarifa normal, tarifa promocional, tarifa para grupos, tarifa com café da manhã incluído etc.

### **Passo inicial obrigatório**

Antes de qualquer configuração no sistema, é necessário que você **crie essa tarifa diretamente nos canais de venda.**

**Onde criar a tarifa:**

**2.Você deve criar a tarifa em cada canal de venda que usa. Os canais comuns são:**

* Booking
* Expedia
* Decolar
* Hbook
* Outros (se houver)

### **3. Informações que precisamos para o mapeamento**

### Após a criação **dos** canais, preciso que **entre** em contato com o suporte para que eles **possam** fazer o mapeamento dessa tarifa.

**4. O que informar:**

### **Em quais canais a nova tarifa já foi criada** (ex.: Booking e Expedia, mas ainda não está no Decolar)

**Em quais categorias essa tarifa está vigente em cada canal**
 (ex.: Categoria Superior – Booking | Categoria Deluxe – Expedia)

**5. Se já existe tarifa mapeada**

* **O que informar:**

Diga se algum canal já tem essa tarifa conectada ao sistema e se você quer copiar os valores dela para os outros canais.

* **Por que isso ajuda?**

Se você já tem a tarifa funcionando em um canal, podemos copiar os valores para os novos canais. Isso economiza seu tempo, pois você não precisa digitar tudo de novo.

**Exemplo de como informar:**

* "Sim, a tarifa já está mapeada no Booking. Por favor, copiem os valores para a Expedia também."

**OU**

* "Não, vou lançar os valores manualmente em cada canal."

**Observação:** O suporte vai fazer o mapeamento no sistema e avisar quando estiver pronto para usar.

|  |  |
| --- | --- |
| **Termo** | **Significado** |
| **Canal de Venda** | Plataforma onde as pessoas fazem reservas. Exemplos: Booking, Expedia, Decolar. |
| **Categoria** | Tipo de quarto. Exemplos: Standard, Superior, Luxo, Deluxe, Suíte. |
| **Copiar / Replicar** | Duplicar valores de um canal para outro para economizar tempo de digitação. |
| **Hbook** | Motor de reservas do hotel. Sistema para reservas diretas pelo site do hotel. |
| **Lançar Manualmente** | Digitar os valores um por um, sem copiar de outro canal. |
| **Mapeamento** | Conexão entre a tarifa criada no canal e o sistema do hotel. |
| **Tarifa** | Preço da hospedagem. Pode ter diferentes tipos: normal, promocional, com café, etc. |
