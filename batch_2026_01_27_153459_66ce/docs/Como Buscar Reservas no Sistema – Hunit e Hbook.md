# **2. Formas de Buscar uma Reserva**

## **Busca pelo Número da Reserva (forma mais direta)**

**3. Se você tem o número da reserva, basta:**

* Ir até o campo **N° da Reserva**
* Inserir o código completo
   ➝ *Exemplo: RES000325-14638*
* Clicar em **Filtrar ou Pesquisar**

**Importante:** A busca só funciona se o número for digitado exatamente como está no sistema (número no canal de origem)

## **4. Buscar pela Data da Reserva**

Use esta opção quando você não sabe o número da reserva, mas sabe **quando ela foi criada**.

1. Clique na flag **Data da Reserva**
2. Informe o **período de início** e **de fim**
3. Clique em **Filtrar**

## **5. Buscar pela Data de Check-in**

Recomendada quando você sabe aproximadamente **para que dia o hóspede chegaria**.

1. Clique na flag **Data do Check-in**
2. Informe o intervalo desejado
3. Clique em **Filtrar**

## **6. Busca pela Data de Check-out**

Útil quando você tem referência de **quando o hóspede deixaria o hotel**.

1. Clique na flag **Data do Checkout**
2. Defina o período de início e fim
3. Clique em **Filtrar**

## **7. Buscar pelo Nome do Hóspede**

Se você sabe o nome do hóspede, mas não tem o número da reserva, é possível buscar assim — porém será necessário ter **ao menos uma ideia do período** (data da reserva, check-in ou checkout), pois a busca pelo nome precisa ser combinada com datas.

1. Digite o **nome do hóspede**
2. Selecione um intervalo de datas (reserva / check-in / checkout)
3. Clique em **Filtrar**

**Dica:** Confira se digitou exatamente igual (Letras maiúsculas, traços)

**Glossário**

|  |  |
| --- | --- |
| **Termo** | **Significado** |
| Número da Reserva | Código único de cada |
| Data da reserva | Dia em que reserva foi Feita Criada |
| Check-in | Data de entrada /Chegada do Hospede |
| Checkout | Data de saída do hospede |
| Filtrar | Botão para aplicar a busca |
| Período | Intervalo de datas (data início até data fim) |
| Hospede | Pessoa que fez a reserva ou vai se hospedar |
