**Como acessar o HBOT**

**1. Acesso ao Hbot**

* **Acesse a URL do HBot:** <https://hbot.hsystem.com.br/login?returnUrl=%2Fapp%2Fchoosecompany>
* **Tela Inicial:** Na tela inicial do **HBot**, clique no botão azul **"LOGIN ÚNICO"**.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_153459_66ce/images/Como_acessar_o_HBOT__image2_3f0eb686.png)

**2. Fazer Login no HBOT**

* Veja o campo Endereço de E-mail
* Digite seu e-mail
* Veja o campo senha (logo abaixo)
* Digite sua senha
* Clique no botão azul ENTRAR

**Esqueceu a senha?**

Se não lembra da senha, clique em ESQUECEU SUA SENHA

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_153459_66ce/images/Como_acessar_o_HBOT__image_0c857346.png)

* **Código de Autenticação (Se você tiver 2 FA)**

Este passo só aparece se você tem autenticação de dois fatores ativada

* A tela vai pedir ‘Código de Autenticador1
* Pegue seu celular
* Abra o aplicativo autenticador (Google Authenticator, Microsoft Authenticator, etc)
* Procure o código de 6 dígitos
* Clique no botão azul Entrar

**Observação:** Se você já está logado em outro sistema (Hbook ou Hunit por exemplo) pode ser que o código NÃO seja pedido. Neste caso, você vai direto para a próxima tela.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_153459_66ce/images/Como_acessar_o_HBOT__image4_ee9f456b.png)

**3.Selecionar a Propriedade**

Este passo só aparece se você gerencia mais de um hotel

* A tela vai mostrar selecione uma propriedade
* Veja a barra de pesquisa (Escrito pesquisar)
* Digite para buscar seu hotel **(Nome do hotel, ID da propriedade (número) ou CNPJ do hotel)**
* Seu hotel vai aparecer na lista
* Clique no nome do hotel

**Basta clicar na Propriedade desejada**

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_153459_66ce/images/Como_acessar_o_HBOT__image3_23883870.png)

**Observação:** Você pode usar os filtros ‘Ativos’ e ‘Inativos’ para ver apenas hotéis ativos e inativos.

**Glossário**

|  |  |
| --- | --- |
| **Termo** | **Significado** |
| Hbot | Sistema de atendimento Inteligente da HSystem |
| Login Unico | Botão para entrar usando mesma conta Hsystem |
| 2FA | Autenticação de dois fatores (senha + código do celular) |
| Propriedade | Hotel ou estabelecimento que você gerencia |
| ID da Propriedade | Número do hotel no sistema |
| Filtros Ativos/nativos | Opções para ver hotéis que estão funcionando ou desativados |
| Sessão | Tempo que você fica logado no sistema |
