**Hunit – Oferta Last Minute**

**Criando ofertas Last Minute no gestor de canais**

## **1. Introdução à Configuração de Ofertas**

Depois de escolher o canal e o tipo de oferta que deseja criar, você chegará à tela de **configuração da oferta**. Esta é onde a mágica acontece - onde você define todos os detalhes precisos que o HUNIT vai enviar automaticamente para o canal de venda escolhido.

Cada canal tem suas particularidades, seus campos específicos e suas regras próprias. O que você pode configurar no HBOOK é diferente do que pode configurar no Booking.com, que por sua vez é diferente do Hoteisnet. Mas não se preocupe - o sistema só mostra as opções que são compatíveis com o canal que você escolheu.

Neste guia, vamos explorar em detalhes como configurar ofertas para cada canal, entendendo cada campo, cada opção, cada filtro. Vamos ver exemplos práticos e estratégias que funcionam no mundo real. Ao final, você será capaz de criar ofertas sofisticadas que maximizam sua receita enquanto atraem o tipo certo de hóspede.

## **Ofertas Last Minute para o Booking**

Neste guia completo, vamos explorar cada detalhe de como configurar ofertas Last Minute especificamente no Booking.com através do HUNIT.

## **Seção: Dados Gerais da Oferta Last Minute**

Vamos agora à configuração técnica passo a passo.

### **Campo: Criando oferta para o canal**

**O que você verá:** "Booking" ou "Booking.com"

Este campo apenas confirma visualmente que você está configurando uma oferta Last Minute para o Booking. É informativo e não editável - apenas uma validação de segurança.

### **Campo: Tipo da oferta no canal**

**O que você verá:** "Oferta de Última Hora", "Last Minute", "Promoção de Última Hora" ou variação similar.

Este campo mostra como o Booking classifica internamente ofertas Last Minute.

**Você não edita este campo** - foi definido quando você escolheu "Last Minute" como tipo de oferta.

### **Campo: Nome da oferta**

Agora sim, o primeiro campo que você preenche! E no caso de Last Minute, o nome é ESPECIALMENTE importante porque precisa transmitir urgência.

**Exemplos de nomes para Last Minute:**

* "Oferta de Última Hora - 30% OFF"
* "Desconto Relâmpago - Reserve Agora"
* "Última Hora - Vagas Limitadas"

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_20_154550_09d9/images/Hunit_-_criando_ofer_image_c6ec3768.png)

**Seção: Tipos de Quartos**

Nesta seção, você define **a quais quartos a oferta será aplicada**.

O sistema lista automaticamente os quartos disponíveis no canal selecionado, como:

* *Standard Double Room*
* *Suíte*
* *Single Room*

Você pode marcar apenas os quartos que devem participar da promoção.
Há também os botões:

* **Marcar Todos:** seleciona todos os quartos de uma vez.
* **Desmarcar:** limpa todas as seleções.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_20_154550_09d9/images/Hunit_-_criando_ofer_image3_5badec20.png)

**Seção: Tipos de Tarifa**

Nesta seção, você define **a quais tarifas a oferta será aplicada**.

O sistema lista automaticamente as tarifas disponíveis no canal selecionado, como:

* *Tarifa Flex c/ Café*
* *Tarifa Flex c/ Café*
* *Tarifa NR (Não reembolsável)*

Você pode marcar apenas as tarifas que devem participar da promoção.
Há também os botões:

* **Marcar Todos:** seleciona todas as tarifas de uma vez.
* **Desmarcar:** limpa todas as seleções.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_20_154550_09d9/images/Hunit_-_criando_ofer_image9_2511fd34.png)

**Condições de Aplicabilidade**

Aqui você define **em que período a oferta será válida** e, se quiser, adiciona **filtros adicionais**.

**Botão “Novo Filtro”**

O botão **“NOVO FILTRO”** serve para configurar as **regras e condições da oferta**.

Lista de opções:

**PERÍODO DA OFERTA**
Defina o período da estadia em que a oferta poderá ser aplicada.

**É UM VISITANTE REGISTRADO**
Defina se a oferta será aplicada somente para visitantes registrados.

**OFERTA SERÁ NÃO REEMBOLSÁVEL**
Defina se a política da oferta será Não Reembolsável.

**ANTEDÊNCIA MÁXIMA**
Defina qual a diferença máxima em dias entre a data da reserva e a data do check-in.

**ESTADIA MÍNIMA**
Defina a quantidade de noites mínimas para que a oferta seja aplicada.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_20_154550_09d9/images/Hunit_-_criando_ofer_imageb_893584dc.png)

**Forma de Desconto**

Essa seção define **como o desconto será aplicado**:

* **Percentual de Desconto:** campo numérico onde você insere o valor do desconto em porcentagem (%).

Exemplo: digite “10” para aplicar 10% de desconto.

* **Aplicar nas seguintes noites:** menu suspenso que permite escolher se o desconto será aplicado em:
  + **Todas as noites** da estadia;

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_20_154550_09d9/images/Hunit_-_criando_ofer_image4_b2a03d71.png)

* + **Dias da semana**, pode selecionar quais dias será aplicado o desconto.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_20_154550_09d9/images/Hunit_-_criando_ofer_image10_79205b4e.png)

Depois de preencher todas as informações, clique em:

* **SALVAR:** grava a oferta e envia para o canal.
* **CANCELAR:** descarta as alterações e retorna à listagem.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_20_154550_09d9/images/Hunit_-_criando_ofer_image12_96157239.png)

Ao salvar, o sistema validará os dados e, se estiver tudo correto, a oferta aparecerá na tela principal de listagem com o status **ATIVO**.

## **Glossário de Termos**

|  |  |
| --- | --- |
| **Termo** | **Definição** |
| **Expedia** | Segunda maior plataforma de reservas online global, forte em mercados americano e asiático. |
| **Expedia Rewards** | Programa de fidelidade onde clientes acumulam pontos e sobem níveis ganhando benefícios. |
| **Weekend Getaway** | Escapada de fim de semana, viagem curta típica de americanos. |
| **Pacote** | Combinação de voo + hotel (às vezes + carro) vendida em conjunto. |
| **Last Minute** | Oferta para reservas feitas com pouquíssima antecedência (1-7 dias antes do check-in). |
| **Mobile-Only** | Oferta exclusiva para quem reserva via app ou navegador mobile. |
| **Flash Sale** | Venda relâmpago de curtíssima duração para criar urgência máxima. |
| **RevPAR** | Revenue Per Available Room. Receita dividida por quartos disponíveis. |
| **Antecedência Máxima** | Número máximo de dias entre reserva e check-in para oferta se aplicar. |
| **Rotatividade** | Frequência de check-ins e checkouts. Alta rotatividade aumenta custos operacionais. |
| **FOMO** | Fear Of Missing Out. Medo de perder uma oportunidade, gatilho psicológico poderoso. |
| **Dispositivo Mobile** | Filtro que limita oferta apenas para buscas feitas em celular ou tablet. |
| **Exceto nas Datas** | Períodos específicos onde oferta não se aplica, mesmo dentro do período geral. |
