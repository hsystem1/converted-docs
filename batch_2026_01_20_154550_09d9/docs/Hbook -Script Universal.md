**Customizar Script Universal**

**Hbook (Motor de Reservas)**

**1. Script Universal (Widget de Reservas)**

**O que é o Script Universal?**

O Script Universal é um código que você coloca no seu site para exibir o formulário de reservas do HBOOK. Pense nele como uma ponte que conecta seu site ao sistema de reservas.

**2. Para que serve?**

Com o Script Universal, seus clientes podem:

• Fazer reservas diretamente no seu site

• Escolher datas de entrada e saída

• Informar número de adultos e crianças

• Usar códigos promocionais

**3. Objetivo da Funcionalidade**

Ele é a **fundação tecnológica** que liga o sistema de reservas HBOOK ao seu website, fazendo com que o widget e todas as outras ferramentas de reserva direta sejam exibidas e operem de forma eficiente.

**Casos de Uso**

**Por que usar essa funcionalidade?**

1. Garantir o Funcionamento Integrado
2. Inclusão Simples e Completa
3. Habilitar a Customização e os Widgets

**Opções de Configuração**

**Opção 1**: Acesse o **HBOOK** e, no menu, clique em **Propriedade>Integrações>Script Universal.**

* Após inserir o código no seu site, volte ao **HBOOK** e **Ative o Script**

**Opção 2: Customizando o Widget e o Formulário**

* **Na mesma página, você pode customizar o seu widget e formulário na seção** Configurações Gerais

**Opção 3: Modelo de Widget:** Você pode escolher entre dois modelos de apresentação:

* **Formulário Horizontal**
* **Formulário Lateral**

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_20_154550_09d9/images/Hbook_-Script_Univer_image8_818b4ce7.png)

**Opção 4:** Configuração do Formulário Horizontal: Ao escolher o formulário horizontal, você terá acesso a algumas configurações adicionais:

* **Bibliotecas:** ative as duas opções disponíveis.
* Exibições: Se desejar que cada campo mostre sua referência, marque a opção **"Exibir label acima dos campos”**

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_20_154550_09d9/images/Hbook_-Script_Univer_image6_cc8e9a74.png)

**Opção 5:** Posição do Widget

* Defina onde o formulário será exibido na página**: Topo ou Rodapé**
* Escolha também onde o **Pop-up** ficará quando o formulário estiver minimizado: **Lado esquerdo** ou **Lado Direito da tela.** Exemplo de **Pop-up;** é possível visualizar como ficará o pop-up com essas configurações

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_20_154550_09d9/images/Hbook_-Script_Univer_image7_0f84dad6.png)

**Opção 6:** Cores

* Tanto no formulário horizontal, quanto no pop-up, você pode definir as cores. Basta inserir o **código Hexadecimal** ou o **nome da cor em inglês** nos campos correspondentes.
* Defina o número máximo de **adultos e crianças** que poderão ser incluídos na busca do formulário.

**IMPORTANTE:** Clique em **Salvar Alterações** para finalizar

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_20_154550_09d9/images/Hbook_-Script_Univer_image5_a1f67c13.png)

**Opção 7:** Formulário Lateral; se optar pelo formulário lateral, a configuração é mais simples;

* Definir **a posição na tela**
* Escolher **a cor de fundo dos ícones.**

**O que o Script Universal Envia para o HBOOK (Variáveis/Parâmetros)**

* Data **Check-in** e **Check-out**
* Quantidade de **Adultos** e **Crianças**
* **Código Promociona**

**Glossário**

|  |  |
| --- | --- |
| **Script Universal** | É um código único que facilita a instalação de diversas ferramentas de rastreio e marketing de uma só vez. Ele simplifica a integração do seu motor de reservas com outros serviços. |
| **HBOOK** | É o nome do seu Motor de Reservas, o sistema onde os clientes realizam as compras de estadias. |
| **Tag de Rastreamento** | Um pequeno pedaço de código que monitora como as pessoas usam o seu site para melhorar suas vendas. |
| **Análise de Dados** | O processo de observar o comportamento dos visitantes para descobrir quais páginas eles visitam e onde desistem da reserva |
| **ROI (Retorno sobre Investimento)** | Uma métrica que ajuda a entender se o dinheiro gasto em anúncios está realmente trazendo lucro através de reservas diretas. |
| **Remarketing** | Estratégia para mostrar anúncios específicos para pessoas que já visitaram seu site, mas não finalizaram a reserva. |
| **Variáveis (Eventos)** | Ações específicas que o sistema identifica para enviar ao rastreador, como uma busca por datas ou a conclusão de uma compra. |
| **Scripts Customizados** | Espaço dentro do sistema para inserir códigos personalizados no cabeçalho ou no final da página do motor de reservas. |
