# **Criação ou Mapeamento de Nova Categoria**

Para realizarmos a criação e o mapeamento de uma nova categoria (acomodação) em nosso sistema e nos canais de venda, precisamos que você nos envie algumas informações essenciais. Essas informações garantem que a categoria funcione corretamente com disponibilidade, tarifas e reservas integradas.

**1. O que é uma categoria**

É o tipo de acomodação que você oferece no hotel

Exemplo:

* Quarto Standard
* Quarto Luxo
* Suíte Premium
* Chale
* Apartamento Duplo

**2. Quando criar categoria**

**Você precisa criar uma categoria quando:**

## **Informações necessárias**

Ao solicitar a criação ou mapeamento de uma categoria, envie:

|  |  |
| --- | --- |
| **Informação** | **Detalhes necessários** |
| Nome da categoria | Nome da categoria em **cada canal** onde será configurada |
| Canais onde já está criada | Informar em quais canais a categoria **já existe.**  Confirmar se já está criada no Booking, Expedia, Decolar, Hbook (ou outros) |
| Quantidade de hóspedes (PAX) | Informar **mínimo e máximo** de ocupantes adultos permitidos. Todos os canais precisão está com a mesma quantidade. |
| PMS (gestor hoteleiro interno) | Informar se possui PMS e se a categoria **já está cadastrada lá** |

**O que você precisa saber antes de começar:**

O suporte da Hsystem NÃO cria categorias nos canais (Booking, expedia, etc.) O suporte faz o MAPEAMENTO (Conecta as categorias já existem)

## **Situações possíveis**

### **2. Categoria já existe em alguns canais e será adicionada a um novo**

Se a categoria já está criada e ativa em alguns canais (ex: Booking e Decolar), mas será incluída agora em outro canal (ex: Expedia), siga este fluxo:

1. **Criar a categoria no novo canal (Ex: Expedia)**
2. Informar o **nome exatamente como aparece nesse canal**
3. Nos informar se deseja que, após o mapeamento:
   1. **Se será enviado os preços manualmente** (você vai lançar os preços), ou
   2. **Replicamos os preços de outro canal** (ex: copiar preços da Decolar para Expedia)
   3. **Replicar além do preço, também as restrições**.

### **3. Categoria ainda não existe em nenhum canal**

* Neste caso, você deve primeiro **criar manualmente nos canais onde deseja vender**.
   Depois, envie os nomes e canais exatos para iniciarmos o mapeamento.

### **4. Categoria existe apenas no PMS**

* Se a categoria existe apenas no seu sistema interno (PMS), é necessário primeiro criar nos canais de venda.
   Após criada, nos envie para iniciar o mapeamento.

## **5.Modelo de informações para envio**

Você pode responder neste formato:

**Categoria:** Suíte Premium
 **Booking:** Premium Suite Double (já existente)
 **Expedia:** Suíte Premium Luxo (a criar/agora criada)
 **Decolar:** Suíte Premium (já existente)
 **Hbook:** Não criada
 **PAX mínimo/máximo:** 1 a 4 hóspedes
 **Possui PMS:** Sim
 **Categoria já criada no PMS?** Sim
 **Configuração de preços no novo canal:** Replicar preços do Booking

## **Sobre a configuração de preços**

Após o mapeamento da categoria no novo canal, você pode escolher:

|  |  |
| --- | --- |
| **Opção** | **O que significa** |
| **Enviar preços manualmente** | Você mesmo irá cadastrar as tarifas diretamente para o canal |
| **Replicar preços de outro canal** | Copiamos a configuração de preços e restrições de um canal onde a categoria já existe |

**Recomendação**

* Se você já tem a categoria funcionando em um canal (Por exemplo, Booking) é mais fácil copiar os preços desse canal para o novo.

## **Observações importantes**

* Não realizamos mapeamento de categorias **não criadas nos canais ou PMS**
* Envie os nomes **exatamente como aparecem nos canais**, sem abreviações
* Informações incompletas podem atrasar o processo
* Após o mapeamento, é recomendável validar disponibilidade e tarifas

**Glossário**

|  |  |
| --- | --- |
| **Termo** | **Significado** |
| Categoria | Tipo de Acomodação (Quarto, Suite, Chalé) |
| Criar | Cadastrar categoria no canal |
| Mapear | Conectar Categoria do Canal com PMS |
| Pax | Quantidade de pessoas que cabem |
| Canal | Site de reserva (Booking, Expedia) |
| PMS | Sistema de Gestão do Hotel |
| Restrições | Regras de venda (Minimo noites, etc..) |
