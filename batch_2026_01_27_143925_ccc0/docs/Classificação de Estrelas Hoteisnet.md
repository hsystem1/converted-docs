# **Classificações de Estrelas do Hotel em Nossos Sistemas Hoteisnet**

**1.Regra Obrigatória:**

Para alterar estrelas no seu sistema, você PRECISA ter certificado oficial do Ministério do Turismo (MTur)

**O que é a classificação**

Classificação por estrelas é categoria oficial do seu hotel

No Brasil, só vale a classificação do governo (Sistema SBClass - Ministério do turismo)

Para garantir a **conformidade e precisão** das informações de todos os meios de hospedagem, utilizamos a **classificação oficial** do governo brasileiro.

## **Requisitos para Alteração da Classificação**

Para que possamos alterar a classificação por estrelas do seu hotel em nossa plataforma, é **obrigatório** que o empreendimento possua a **certificação oficial** emitida pelo **Ministério do Turismo (MTur)**, através do **Sistema Brasileiro de Classificação de Meios de Hospedagem (SBClass)**.

* **Não podemos** aceitar classificações autoavaliadas ou de fontes não oficiais.
* A classificação deve ser comprovada com o **documento oficial e válido** emitido pelo MTur.

## **2. O Processo Oficial de Classificação (SBClass)**

A classificação por estrelas no Brasil é um processo formal e regulamentado, garantindo um padrão nacional de qualidade.

|  |  |
| --- | --- |
| **Estrelas** | **Categoria Oficial** |
| **★** | Simples (1 Estrela) |
| **★★** | Econômica (2 Estrelas) |
| **★★★** | Turismo (3 Estrelas) |
| **★★★★** | Superior (4 Estrelas) |
| **★★★★★** | Luxo (5 Estrelas) |

### **3. Como Obter ou Atualizar Sua Classificação Oficial:**

1. **Adesão ao SBClass:** O hotel deve se cadastrar na plataforma do **Ministério do Turismo**.
2. **Avaliação:** O processo envolve o envio de documentação e uma **inspeção técnica presencial** realizada por avaliadores credenciados.
3. **Certificado:** Ao atender aos requisitos, o hotel recebe o certificado oficial de classificação.
4. **Nós NÃO aceitamos:**

* Auto - avaliação
* Classificação de site particulares
* Certificado vencido

**5. Já tenho certificado – O que fazer?**

**Se você já tem o certificado valido**

* Pegue o certificado (Documento em PDF ou Foto)
* Entre em contato com o suporte@hsystem.com.br
* Anexe o certificado no e-mail
* Suporte vai atualizar no sistema
* Você recebe a confirmação

**6.Onde buscar ajuda**

**ABIH Pode ajudar:**

A Associação Brasileira da indústria de Hotéis (Abih) oferece orientação e apoio técnico para auxiliar os hotéis no processo de classificação.

Eles podem;

* Explicar o processo completo
* Ajudar com documentação
* Orientar sobre adequações necessárias
* Preparar hotel para inspeção

**Glossário**

|  |  |
| --- | --- |
| **Termo** | **Significado** |
| SB Class | Sistema Brasileiro de Classificação de Meios de Hospedagem |
| MTur | Ministério do Turismo (Governo Federal) |
| ABIH | Associação Brasileira da Industria de Hotéis |
| Certificado Oficial | Documento emitido pelo MTur que comprova as estrelas |
| Avaliador Credenciado | Pessoa autorizada pelo MTur para fazer inspeção |
