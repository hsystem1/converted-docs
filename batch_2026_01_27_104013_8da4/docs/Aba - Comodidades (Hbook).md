**Comodidades**

**Hbook (Motor de Reservas)**

**O que são comodidades?**

São todos os serviços e facilidades que o hotel oferece aos hospedes. Por exemplo: Piscina, Wi.fi, Café da manhã, estacionamento, academia etc.

**1. Aba de Comodidades:**

* Essa tela corresponde à configuração de comodidades do hotel, que permite selecionar quais serviços e facilidades estão disponíveis para os hóspedes no motor de reservas.
* As comodidades são serviços ou produtos que o seu hotel oferece ou facilita para o hóspede, podendo ter custo extra ou estarem inclusos na diária.
* No menu principal, clique em “Propriedade,” depois em “Geral” e selecione a aba “Comodidades.”

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_104013_8da4/images/Aba_-_Comodidades_%28H_image_6afe37e4.png)

**2. Marque tudo o que seu hotel oferece:**

* Id: identificação interna de cada item.
* Disponível? botão seletor (ativar/desativar) para indicar se a comodidade deve ser exibida ao hóspede.
* Comodidades: lista organizada por categorias, como:
* Áreas de lazer: exemplo – beira-mar, piscina infantil, ofurô.
* Internet: exemplo – Wi-Fi gratuito disponível em toda a propriedade.
* Atividades: exemplo – jogo de dardos, aula de yoga, brinquedoteca.
* Alimentos e Bebidas: exemplo - Bar, Máquina de lanche, restaurante, café da manhã.
* Gerais: exemplo - Jacuzzi Aquecida, Aquecimento, aceita pets, Voltagem 220V.
* Serviços: exemplo - Recepção 24 horas, Lavanderia, Concierge, Cadeiras de praia e guarda-sol disponível para uso gratuito dos hóspedes.
* Idiomas falados: exemplo - Português, Espanhol, Inglês.

**3. Como funciona:**

* O hotel pode marcar apenas as comodidades que realmente oferece.
* Essas informações aparecem no motor de reservas, ajudando o hóspede a conhecer melhor a estrutura antes da reserva.
* A organização por categorias facilita a visualização e seleção.
* Possui o botão **“Mostrar apenas comodidades selecionadas”**, que filtra a lista exibindo somente os itens já ativados. Isso facilita a conferência do que será apresentado no motor de reservas, evitando que o hotel precise percorrer toda a lista.
* **Campo de filtro:** permite pesquisar comodidades específicas dentro da lista, agilizando a configuração.

**Glossário**

|  |  |
| --- | --- |
| **Termo** | **Significado** |
| **Aba** | Seção dentro de uma tela. Como páginas de um caderno. |
| **Categoria** | Grupo. Conjunto de itens semelhantes organizados juntos. |
| **Comodidade** | Serviço ou facilidade oferecida pelo hotel. Exemplo: Wi-Fi, piscina. |
| **Concierge** | Serviço de assistência ao hóspede. Ajuda com reservas, passeios, informações. |
| **Filtro** | Ferramenta de busca. Mostra apenas o que você procura. |
| **HBOOK** | Motor de Reservas. Sistema de reservas online do hotel. |
| **Id** | Número de identificação. Código único de cada item. |
| **Ofurô** | Banheira de hidromassagem. Jacuzzi com água quente. |
| **Propriedade** | Termo usado para se referir ao hotel no sistema. |
