**Geral**

**Hbook (Motor de reservas)**

**1. O que são as configurações gerais?**

* As Configurações Gerais são onde você cadastra todas as informações essenciais do seu hotel no sistema HBook. É como criar o cadastro completo da sua propriedade, incluindo dados básicos, fotos, regras, formas de pagamento e muito mais.

**2. Como acessar:**

Clique em PROPRIEDADE → GERAL.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_104013_8da4/images/Aba%C2%A0_-_Geral_%28Hbook%29_image_218a40e1.png)

**3. Entendendo as abas de configurações:**

O processo é dividido em etapas que organizam os principais dados:

* **Dados Gerais:** Informações básicas do hotel (nome, endereço, telefones)
* **Fotos:** Imagens do hotel, logo e galeria
* **Política Geral:** Regras da propriedade (animais, fumo, check-in/out)
* **Privacidade:** Como os dados dos hóspedes são protegidos
* **Comodidades:** Recursos oferecidos (piscina, Wi-Fi, estacionamento)
* **Políticas de Idade:** Regras para crianças e valores
* **Meios de Pagamento:** Formas de pagamento aceitas
* **Taxas:** Impostos e cobranças adicionais

**Importante!**

* Sempre que for editar uma aba da configuração geral do hotel, é necessário salvar as informações de cada aba antes de mudar para outra.
* Por exemplo: se você está editando o texto descritivo na aba Dados Gerais, antes de acessar a aba meios de pagamento é preciso salvar aquela tela, caso contrário as alterações feitas serão perdidas.
* Essa regra vale para todas as abas do cadastro do cadastro geral do hotel.

**4. Aba Dados gerais:**

* No menu principal lateral direito, clique em **PROPRIEDADE**, depois clique em **GERAL.**

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_104013_8da4/images/Aba%C2%A0_-_Geral_%28Hbook%29_imagea_f2765679.png)

**5. Identificador:**

* **C**ódigo único do hotel dentro do sistema, gerado automaticamente.
* **Ver motor de reserv**a - Link que leva para a tela pública onde hóspedes fazem reservas

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_104013_8da4/images/Aba%C2%A0_-_Geral_%28Hbook%29_image5_77b073fe.png)

**6.Nome da Propriedade:**

* Nome do hotel que aparece para o público. Para mudar, entre em contato com o suporte.

**7.Descrição completa da propriedade**

* Aqui você conta sobre sua propriedade, seus diferenciais e localização. Você pode escrever em três idiomas (português, inglês e espanhol) clicando nas bandeirinhas.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_104013_8da4/images/Aba%C2%A0_-_Geral_%28Hbook%29_image7_a5f8ecf5.png)

**8. URL Página Hotel:** URL Página da propriedade

* Link para o site oficial do hotel. Ao clicar na logo no motor, direciona para o site. Lembre-se de incluir https:// ou http:// antes de endereço.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_104013_8da4/images/Aba%C2%A0_-_Geral_%28Hbook%29_image9_ea001422.png)

**9. Telefones de Contato**

* Telefone Principal: Número principal do hotel
* Telefone Secundário: Outro número de contato
* WhatsApp: Número do WhatsApp para contato

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_104013_8da4/images/Aba%C2%A0_-_Geral_%28Hbook%29_image4_0b0834ed.png)

### **10. Endereço da propriedade:**

Preencha todos os campos do endereço:

* • CEP
* • Logradouro (nome da rua)
* • Bairro
* • Cidade
* • Estado
* • País

**URL no Google Maps:** Link da localização no Google Maps

Abra o Google Maps, procure seu hotel, clique em 'Compartilhar' e cole o link aqui.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_104013_8da4/images/Aba%C2%A0_-_Geral_%28Hbook%29_image3_52605ace.png)

**11. Design do Motor de Reservas (Opcional):**

Aqui o hotel pode configurar a identidade visual do motor. Recomenda-se que apenas pessoas com conhecimento em web design2

* Cor de Fundo (Logo), Cor de Fundo (Background), Cor para Mobile, Cor Principal: permitem ajustar a paleta de cores (precisa preencher com o código da cor).
* Opções de exibição:
* Centralizar logo
* Mostrar sombra no logo (Box-Shadow)
* Exibir cabeçalho (Cover)
* Exibir informações do hotel no rodapé
* Mostrar o melhor preço do dia no calendário
* Mostrar valores já com ofertas aplicadas

Essas opções ajudam a personalizar a experiência do usuário, mantendo o motor alinhado à identidade visual do hotel.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_104013_8da4/images/Aba%C2%A0_-_Geral_%28Hbook%29_image2_a3ab9738.png)**12. Texto para Sem Disponibilidade:**

* O hotel pode configurar a mensagem exibida quando não há disponibilidade de quartos.

**13. Idiomas disponíveis:**

* Marque os idiomas que ficarão disponíveis em tela pública do motor para o hóspede escolher. Lembre-se o sistema não faz tradução automática — portanto, se o hotel deseja que o texto apareça em inglês ou espanhol, você mesmo deve inserir a versão traduzida.

**Glossário**

|  |  |
| --- | --- |
| **Termo** | **Significado** |
| Aba | Seção ou página dentro de um sistema. Como as abas de uma pasta ou arquivo. |
| Comodidades | Recursos e facilidades oferecidos pelo hotel, como piscina, Wi-Fi, estacionamento, café da manhã, academia, etc. |
| HBook | Nome do sistema de gerenciamento de reservas usado pelo hotel. |
| Hóspede | Pessoa que se hospeda ou faz reserva no hotel. Cliente do hotel. |
| Identificador | Código único que identifica o hotel dentro do sistema. É gerado automaticamente. |
| Logotipo (Logo) | Símbolo ou imagem que representa a marca do hotel. |
| Política | Regras e normas estabelecidas pelo hotel. Exemplo: política de cancelamento, política de animais. |
| Propriedade | O hotel em si. Também pode ser chamado de estabelecimento. |
| Rodapé | Parte inferior de uma página de site. Geralmente contém informações de contato e links importantes. |
| Taxa | Valor adicional cobrado além da diária. Exemplo: taxa de serviço, taxa turística. |
| URL | Endereço de um site na internet. Exemplo: <https://www.meuhotel.com.br> |
