**Como configurar Extras**

Hbook

**1. O que são extras?**

**Extras são itens adicionais** que você pode oferecer aos seus hóspedes durante a estadia. Eles podem ser incluídos gratuitamente na diária ou vendidos separadamente.

Exemplos de extras:

* Café da manhã adicional
* Transfer para o aeroporto
* Estacionamento
* Passeios turísticos
* Massagem ou spa

**2. Para que serve?**

A seção Extras tem como objetivo permitir a criação, edição e organização de itens adicionais que podem ser incluídos na diária de hospedagem. Nesta tela, o extra pode ser criado para adicionar a uma tarifa, integrando-se ao valor da diária ou para venda avulsa, podendo ser oferecido e cobrado separadamente da tarifa, fica disponível para o hóspede turbinar sua reserva. Para venda avulsa, o extra precisa ter um valor definido; caso contrário, não será possível vendê-lo separadamente.

**3. Como acessar?**

* **Propriedades -> Extras**

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_image5_c9e99d88.png)

**4.Entendendo a tela de Extras**

No topo, há um campo **“Ordenar Por”**, que permite organizar a lista de extras.

**Colunas:**

* **Código:** Número identificador do extra dentro do sistema.
* **Nome:** Nome do extra, que será exibido aos usuários e hóspedes.
* **Dias Vigentes:** Indica os dias em que o extra está ativo ou disponível.
* **Preço:** Valor do extra. Caso não tenha valor preenchido, não reflete para venda (caso vendido a parte e não embutido na tarifa).

**5. Botões de ação na última coluna**:

* • Ícone de lápis: Para editar as informações do extra
* • Ícone de lixeira: Para excluir o extra
* • Ícone de engrenagem: Para ajustar a ordem de exibição

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_imaged_44b41f06.png)

**6. Opções de configuração**

* Para criar um extra basta ir em PROPRIEDADE → EXTRAS – no canto inferior direito, possui um botão roxo com um +.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_image4_14af92a4.png)

**7.Como criar um Extra incluído na tarifa:**

Preencha as informações:

* **Nome do Extra:** exemplo: Café da manhã adicional, Transfer, Estacionamento.
* **Texto descritivo:** detalhe o que está incluso no extra e informe pontos importantes que o hóspede precisa saber.
* **Valores:** Deixe o valor em R$0,00, já que será incluso.
* **Foto:** Permite adicionar uma imagem ilustrativa do extra.

**Importante:** Mas primeiro preencher todas as informações e salvar o registro. Somente após salvar o extra, no botão roxo no canto inferior direito, é que será possível adicionar uma foto.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_image10_fcb61618.png)

* Para isso, localize o extra criado, clique em Editar e inclua a imagem desejada, clicando no botão **“Adicionar Foto”**, nas dimensões de 170px x 110px, por fim, clicar no botão de salvar (botão roxo no canto inferior direito).

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_image13_82f89433.png)

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_image6_be3164b0.png)

* **Períodos:** Define a disponibilidade do extra por datas e dias da semana:
* **Início e Fim:** seleção de datas em calendário para determinar o período de vigência.
* **Dias da semana:** Checkboxes para indicar quais dias da semana o extra estará disponível (Domingo a Sábado).
* Botões de adicionar (+) ou excluir (lixeira) permitem criar múltiplos períodos de disponibilização ou eliminar um período.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_image7_e5b63511.png)

**Importante:** Essas regras garantem que o hóspede visualize apenas os extras que realmente estão disponíveis para as datas da reserva.

**8.Tipos de Quartos:** Permite selecionar quais tipos de quarto poderão ter o extra:

* A lista apresenta os tipos de quartos cadastrados, permitindo múltipla seleção.
* O extra será apresentado somente nos tipos de quartos selecionados.
* Possui um botão de marcar todos ou desmarcar.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_imageb_b9d4321c.png)

**9.Vincular o Extra na tarifa**

* Após criar ou editar o seu extra, é necessário vinculá-lo corretamente para que ele apareça como um benefício sem custo adicional para o hóspede.

**Para isso, siga estes passos:**

* Acesse o menu Propriedade - Tarifas

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_imagec_138fa08f.png)

* Escolha a tarifa que deseja e clique em editar (última coluna, símbolo de edição)

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_imagea_89068049.png)

* Acesse a aba Extras Inclusos

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_image12_005b4bac.png)

**10. Selecione o extra que deve estar incluído sem custo adicional**

* Cada extra aparece em formato de caixa de seleção (checkbox), permitindo que o hoteleiro escolha quais serão incorporados à tarifa.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_image_a4a620ea.png)

11. Por fim, **clique no botão roxo**, no canto inferior direto, para salvar a edição.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_image2_52cc1dad.png)

**12. Criando extra avulso venda Avulsa e Preencha os seguintes campos:**

* **Nome do Extra:** exemplo: *Café da manhã adicional, Transfer, Estacionamento.*
* **Texto descritivo:** detalhe o que está incluso no extra e informe pontos importantes que o hóspede precisa saber.

**Configure os Valores**

Valores: Permite configurar como o preço do extra será calculado e o preço unitário:

**Por Noite ou por Reserva:**

* Reserva: o valor é cobrado uma única vez, independentemente do número de noites.
* Noite: o valor é cobrado para cada noite da estadia.

**Por Quarto ou Por Pessoa:**

* **Quarto:** o valor é cobrado por quarto.
* **Pessoa**: o valor é cobrado por pessoa. Ao selecionar a opção “Por Pessoa”, são exibidas caixas adicionais para definir se o extra será cobrado por adultos e/ou por criança.

**Preço unitário:**

* Campo numérico para definir o valor do extra.

**Foto:** Permite adicionar uma imagem ilustrativa do extra. Mas primeiro preencher todas as informações e salvar o registro. Somente após salvar o extra, no botão roxo no canto inferior direito, é que será possível adicionar uma foto.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Hbook_-_Extras_image3_99927266.png)

**17. Como organizar a ordem dos Extras:**

* Você pode escolher a ordem em que os extras aparecem para o hospede:

### **Organização Automática**

No topo da tela, há o campo 'Ordenar Por' com três opções:

* **Menor Preço:** Mostra os extras mais baratos primeiros
* **Alfabética (A-Z):** Organiza por ordem alfabética do nome
* **Manual:** Você escolhe a ordem que quiser

### **Organização Manual**

Se você escolher 'Manual', poderá reorganizar da seguinte forma:

Clique no ícone de engrenagem (última coluna) do extra que quer mover

Escolha uma opção:

• Ordenar em primeiro: Move para o topo da lista

• Ordenar antes: Coloca antes de outro extra

• Ordenar depois: Coloca depois de outro extra

• Ordenar em último: Move para o final da lista

**Importante:** A ordem que você define aqui será a mesma que o hospede vera no sistema de reservas.

![](data:image/png;base64...)

**18. Como configurar Extras em vários idiomas:**

Se o seu hotel recebe hóspedes estrangeiros, você pode oferecer os extras em outros idiomas:

Ao criar ou editar um extra, você verá três bandeiras no topo:

• **Bandeira do Brasil:** Português

• **Bandeira dos EUA**: Inglês

• **Bandeira da Espanha:** Espanhol

**Importante:** O sistema não traduz automaticamente! Você precisa escrever o texto em cada idioma que deseja oferecer.

**19.** Quando o hóspede acessar o site, ele poderá escolher o idioma e verá as informações que você escreveu para aquele idioma.

**Glossário**

|  |  |
| --- | --- |
| **Termo** | **Significado** |
| Extra | Item adicional que pode ser oferecido ao hóspede, como café da manhã, transfer, estacionamento etc. |
| HBook | Nome do sistema de reservas que você está usando (também chamado de Motor de Reservas) |
| Tarifa | Preço ou plano que define quanto custa a hospedagem em determinadas condições |
| Extra Incluído | Extra que já vem com a tarifa, sem custo adicional para o hóspede |
| Extra Avulso | Extra que é vendido separadamente, com preço próprio |
| Venda Avulsa | Quando algo é vendido separadamente, não incluído no pacote principal |
| Diária | Valor cobrado por dia de hospedagem |
| Vigente | Que está em vigor, ativo, válido |
| Período | Intervalo de tempo (datas) em que algo está disponível ou ativo |
| Por Noite | Cobrança feita para cada dia de hospedagem |
| Por Reserva | Cobrança feita uma única vez, independentemente do número de noites |
| Por Quarto | Cobrança feita por cada quarto reservado |
| Por Pessoa | Cobrança feita para cada hóspede |
| Tipo de Quarto | Categoria de acomodação (exemplo: Standard, Luxo, Suíte) |
| Ordenar | Colocar em ordem, organizar a sequência |
| Código | Número de identificação único que o sistema usa internamente |
| Motor de Reservas | Sistema que gerencia as reservas do hotel (neste caso, o HBook) |
| Bandeira | Ícone de país usado para selecionar o idioma (Brasil = Português, EUA = Inglês, Espanha = Espanhol) |
| Checkbox | Caixa de seleção que você marca (✓) ou desmarca |
| Aba | Seção clicável que mostra diferentes partes de uma mesma tela |
| Botão Roxo | Botão de cor roxa usado para salvar alterações no sistema |
