**Fotos Gerais**

**Hbook (Motor de Reservas)**

**1. Aba para definir fotos gerais do hotel**

* As fotos são muito importantes! Elas deixam seu motor de reservas mais bonito e profissional.
* No menu principal, clique em “Propriedade” depois “geral” e selecione a aba “fotos”.

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Aba_-_Fotos_Gerais_%28_image6_b017d768.png)

**2.Favicon**

**O que é:** É aquela imagem pequena que aparece na aba do navegador, ao lado do nome do site.

* **O tamanho recomendado é** de 32 pixels por 32 pixels.
* Botão para incluir imagem “Definir favicon”
* Caso queira remover, tem o botão “Limpar favicon”

**3. Logo**

* **O que é:** É a marca do seu hotel que aparece no topo do site de reservas.
* Definir logo: é um elemento gráfico fundamental que representa a identidade visual de uma empresa, marca ou produto, combinando cores, formas e tipografia para comunicar valores e personalidade.
* O tamanho recomendado é de 150px por 120px
* Botão para incluir imagem “Definir logo”
* Caso queira remover, tem o botão “Limpar logo”

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Aba_-_Fotos_Gerais_%28_image7_2e77444e.png)

**4. Foto de Capa**

* O que é: É a primeira imagem grande que os visitantes vêm ao entrar no site de reservas.
* Adicionar nova foto de capa: a capa é a primeira coisa que os visitantes veem ao acessar o motor de reserva.
* O tamanho recomendado é de 1254px por 340px
* Botão para incluir imagem “Adicionar nova foto de capa”
* Caso queira remover, tem o botão “Remover”

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Aba_-_Fotos_Gerais_%28_image4_99ff49a1.png)

### **5. Galeria de Fotos**

* Adicionar novas fotos na galeria: Fotos de apresentação da propriedade.
* O tamanho recomendado é de 800px por 600px ou 900px por 500px
* Botão para incluir imagem “Adicionar novas fotos na galeria”
* Caso queira remover, tem o botão “Remover”
* Para alterar a ordem das imagens, basta arrastar cada foto para a posição desejada e, em seguida, clicar em “Salvar Ordem”

**Outras opções:**

• Baixar todas as imagens: Salva todas as fotos no seu computador

• Excluir todas as imagens: Remove todas as fotos de uma vez

![Imagem](https://raw.githubusercontent.com/hsystem1/converted-docs/main/batch_2026_01_27_144114_87c5/images/Aba_-_Fotos_Gerais_%28_image3_fb034733.png)

* Por fim, clique no botão roxo, no canto inferior direto, para salvar.

**Glossário**

|  |  |
| --- | --- |
| Termo | O que significa? |
| Favicon | O ícone minúsculo que fica na aba do navegador, ao lado do nome do site. |
| Logotipo (Logo) | A imagem que representa o nome e a marca visual do seu hotel |
| Foto de Capa | A imagem principal e maior que fica no topo do site de reservas. |
| Galeria | O conjunto de fotos que mostra os detalhes e áreas do seu hotel. |
| Pixels (px) | É a unidade de medida do tamanho das imagens na internet. |
| Motor de Reservas | O sistema (Hbook) onde o cliente escolhe a data e reserva o quarto. |
